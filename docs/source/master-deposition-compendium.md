## METHODISCHE GRUNDSÄTZE

### Für Depositions (Gerichtliche Befragungen)
1. **Drei-Block-Prinzip**: Eröffnung (offen) → Kerntransaktionen (dokumentenbasiert) → Fallen (konfrontativ)
2. **Dokumentenbasierung**: Jede Falle-Frage referenziert ein konkretes Dokument [DOKUMENT]
3. **Ermittlungsziel**: Jede Frage hat ein explizites Ziel [ZIEL]
4. **Eskalation**: Von allgemein → spezifisch → konfrontativ
5. **Keine Suggestivfragen**: Zeuge soll sich selbst widersprechen
6. **Schweigepausen**: Nach kritischen Fragen Zeit lassen

### Für Journalistische Interviews (SPIEGEL-Methodik)
1. **Vier-Kategorien**: Geschichte (Vertrauen) → Fakten (Boden) → Widersprüche (Konfrontation) → Zukunft (Abschluss)
2. **Taktische Anmerkungen**: [TAKTIK] für den Interviewer
3. **Eskalation**: Niemals sofort konfrontativ — Geschichte zuerst
4. **Dokumente als stumme Zeugen**: Vorlegen ohne Ankündigung
5. **Schweigen als Werkzeug**: Nach harten Fragen nichts sagen
6. **Presserechtliche Prüfung**: Alle Widerspruchs-Fragen müssen faktisch belegbar sein

### FAKT vs. HYPOTHESE
Alle Fragen beruhen auf dokumentierten Fakten, sofern nicht explizit als [HYPOTHESE] gekennzeichnet. Keine Frage enthält unbelegt-spekulative Elemente.
